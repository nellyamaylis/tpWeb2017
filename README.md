TP Web : Javascript et HTML5

Le but du TP a été de construire à la main une application Web suffisamment complexe pour nécessiter la mise en place d’un MVC côté client. 

Il consiste à créer une application Web pour faire du dessin vectoriel, pour dessiner des rectangles, des lignes, ainsi que définir leur couleur et leur épaisseur de trait.
Pour sa réalisation, nous avons utilisé Javascript avec des balises spécifiques permettant de dessiner différentes formes comme var canvas = document.getElementById('myCanvas'); avec canvas déclaré comme une variable.
